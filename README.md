# Python

1. Python Foundations
File: python basics.ipynb

Concepts: Introduction to Python syntax and logic.

Key Highlights: Input/Output operations, variable declaration, and working with primitive data types like integers, floats, and strings.

2. Statistical Analysis & EDA
File: mean-median-mode-outliers.ipynb

Concepts: Descriptive statistics and data quality assessment using a wine dataset.

Key Highlights: Calculating central tendency (Mean, Median, Mode), dispersion metrics, and visualizing data distributions using Box Plots and Correlation Heatmaps to identify outliers.

3. Feature Engineering
File: feature engg-titanic dataset.ipynb

Concepts: Data preprocessing for Machine Learning using the Titanic dataset.

Key Highlights: Handling missing values, categorical encoding (One-Hot Encoding), feature scaling, and dropping redundant columns to optimize model performance.

4. Matrix Factorization (Linear Algebra)
File: matrix factorization.ipynb

Concepts: The mathematical foundations of modern data science.

Key Highlights: Implementing advanced matrix decomposition techniques including LU Decomposition, QR Decomposition, and Singular Value Decomposition (SVD) using NumPy.

Tech Stack Used: * Languages: Python 3.x

Libraries: Pandas, NumPy, Scipy, Seaborn, Matplotlib, ReportLab
